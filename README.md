## Engineering Principles

Good software is not just working code — it's maintainable, predictable, and built to scale.

- **Clean architecture** — service layers, action classes, repository pattern
- **Security first** — validation, authorization, input sanitization at every layer
- **Performance-aware** — query optimization, caching strategies, lazy loading
- **Scalable by design** — database normalization, queue-based processing
- **Honest estimates** — I scope work carefully and communicate blockers early
